data-snippet.io

A service for keeping short pieces of information, snippets, in sync across websites, reglardless of how or where the website is hosted. The snippets can be edited by their owner straight from any page where they are displayed.

It works by including a small library, the snippet loader, on the site where snippets are attached, and using the data-snippet attribute to specify which snippet to show in that element. Snippets are loaded at "view-time". Try it now, for free :)

Herokumon

Herokumon is my final project in GA. It consists of two parts, including the server and UI. It monitors a Heroku application in real-time by attaching as a log drain and visualising the log data using D3.js. The visual shows the number of dynos an application has, with request status breakdown per dyno and response latency per request for the application as a whole.
